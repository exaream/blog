# Golang Installation

## Release history

- [Release History](https://go.dev/doc/devel/release)

## Install old version

```shell
$ go version
go version go1.18.2 darwin/amd64
$ go install golang.org/dl/go1.17@latest
$ go1.17 download
$ go1.17 version
go version go1.17 darwin/amd64
```
