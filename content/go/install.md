# Golang latest version

## Install old version
https://go.dev/doc/tutorial/generics

```shell {hl_lines=[3,4,6]}
$ go version
go version go1.18.2 darwin/amd64
$ go install golang.org/dl/go1.17@latest
$ go1.17 download
$ go1.17 version
go version go1.17 darwin/amd64
```
