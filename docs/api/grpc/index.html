RPC とは #

  • Remote Procedure Call
    • 直訳すると「遠隔手続き呼び出し」
    • あるプログラムがネットワーク上の異なる場所に配置されたプログラムの処理を呼び出して実行すること
    • サン・マイクロシステムズ社がNFS(分散ファイルシステム)の一部で使用するために1976年に開発
